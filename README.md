## 线路这件事，必须认真讲

买 VPS 做什么不重要，线路回程才是核心。

搬瓦工 DC1 机房的三网回程是这样的：

- **电信**：CN2 GIA（AS4809）直连，延迟低、晚高峰波动极小
- **联通**：CMIN2 或 Premium 高级线路，质量好于普通 163
- **移动**：CMIN2 直连，回程经上海，整体表现稳定

有人问：联通走 CMIN2 还是不是走上海？会不会延迟偏高？

根据实测，联通走 CMIN2 经上海回程会比电信高出几十毫秒，这是线路架构决定的，不算 bug，算正常表现。电信用户体验最优，移动次之，联通稍逊但依然是优化线路的水平。

对比同价位来看，这个配置真的很卷。
